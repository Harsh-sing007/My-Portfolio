# Harsh Singh | Portfolio

A stunning, modern personal portfolio built with a focus on immersive user experience, cinematic animations, and a premium dark aesthetic.
https://harshsingh.onrender.com/

## 🚀 About Me
I am a **Cloud Computing student** passionate about **DevOps**, automation, and scalable cloud infrastructure. I enjoy learning how modern applications are deployed and managed using cloud platforms, containerization, and CI/CD pipelines.

## ✨ Key Features
- **Cinematic Experience**: Immersive dark theme with glassmorphism, parallax effects, and smooth scroll-triggered animations.
- **Dynamic Projects Section**: Interactive project cards with 3D-like parallax images and detailed tech tags.
- **Education Timeline**: A modern, visual representation of my academic journey.
- **Credential Showcase**: Integrated certifications section with issuer logos and direct verification links.
- **Interactive Contact Form**: Custom-built glassmorphic communication interface.
- **Responsive Design**: Fully optimized for all devices from mobile to ultra-wide displays.

## 🛠️ Tech Stack
- **Frontend**: React.js
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Styling**: Tailwind CSS / Vanilla CSS
- **Tools**: Vite, Git

## 📂 Project Structure
```text
src/
├── assets/         # Project images and profile media
├── components/     # Reusable UI sections (Hero, About, Projects, etc.)
├── data/           # Structured portfolio data for easy updates
└── main.jsx        # App entry point
```

## 🛠️ Installation & Setup
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
