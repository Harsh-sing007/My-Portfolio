I believe software should solve real human needs. Let me show you how:

*   **Rat in a Maze:** I went deeper into logic with this pathfinding visualizer. It’s one thing to read about an algorithm; it’s another to see it find its way through a maze in real-time using **Backtracking**. 
*   **Production-Style CI/CD Pipeline:** This is where my passion for DevOps truly shines. Using **Docker and GitHub Actions**, I integrated automated security scanning with **CodeQL**. It ensures that every line of code is tested and secure before it ever reaches a user.
